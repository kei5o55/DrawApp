# DrawApp
- C言語とSDL2を用いたシンプルなペイントアプリです．
- マウス入力による描画処理，画像保存機能の実装を通しデスクトップアプリ開発の基礎を学習しました．

# 制作目的
- Githubを実運用し，バージョン管理の基礎を身につける
- C言語によるGUIアプリケーション開発の全体フローを体験


# 主な機能
- マウスクリックによるペイント描画
- 消しゴム機能
- bmp形式での画像保存

# 仕様技術
- C言語
- SDL2

# 工夫
 描画処理
- マウスの座標取得をフレーム毎に処理し，なめらかに線が引ける描画ロジックを設計しました．
　マウスのドラッグによる連続描画時に線が途切れてしまう問題を解決するため，前後フレームの座標間の補完処理を実装しました．

描画補助UIの実装
- 実キャンバス上に，描画箇所のみを表示するオーバーレイ用キャンバスを設計・実装しました
　クリックしていない状態でもマウス下部に描画領域を表示することで，操作性の向上を図っています．
  履歴管理
- 描画履歴を配列で管理し，Undo / Redo 機能の基礎実装を行いました．

# AI活用について
設計検討や不明点の整理においてChatGPTを壁打ちツールとして活用しました．
実装・設計判断・動作検証はすべて自分で行い，
AIの提案内容は検証の上で取捨選択しています．
